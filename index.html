Intro to the Web

All of the files that make up the web are generally stored on servers. To access these files, we use our computers to connect to the internet. Installed on these computers are browsers which send HTTP requests to the internet (ie. world-wide network of computers). The requested files the computer receives is then interpreted and displayed to us on the browser. Below is artist and Reddit co-founder Steve Huffman’s beautiful rendition of this process.

HTML

Elements and Tags are a part of the html programming language. Tags can create or modify content such as text or images. An element is the combination of a tag, and its related content where applicable.

Inline vs Block

Most HTML elements are 'inline' or 'block' elements. Inline elements are typically elements that are displayed on the same line, without line breaks. Block elements will be displayed on a different line or partitioned area. Examples are listed below.

Inline Element Types

<a>
<b>
<br>
<span>

Block Element Types

<p>
<div>
<form>

Lesson 3 Notes

Use CSS to avoid repetition. Don’t forget to keep a logical order to keep the structure need and tidy to prevent mistakes, or make it easier to spot them in the future!

Avoid Repitition

This is key in software development and good practice for any kind of programming. When avoiding repition, unnecessary effort is made and future adjustments to the code may prove difficult. Instead, utilize the DRY (Don't Repeat Yourself) principle. When applied successfully, modifying a single element in your 'system' will not modify other elements; also, modifying element systems will change all the related elements predictably and uniformly.

Three Ways to Add Style

  1. In the HTML document
    1. Provide the style information in the <head> section within <style> tags
  2. In a separate .css file
    1. Reference the separate sheet in the <head> section of your html document, type: <link rel=”stylesheet” type=”text/css” href=”filename.css”>
  3. Inline, within the HTML tags
    1. For example, <div style=”background-color: red; color: white> Useful to style unique parts, but generally avoid using this method

Comments In Program

HTML: <!-- COMMENT -->

CSS Property Inheritance

Inheritable: color, font, letter-spacing, line-height, list-style, text-align, text-indent, text-transform, visibility, white-space and word-spacing

Non inheritable: background, border, display, float and clear, height and width, margin, min- and max-height and -width, outline, overflow, padding, position, text-decoration, vertical-align and z-index
